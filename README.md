# Rakesh Manjhi - Portfolio

Welcome to my UX/UI Design portfolio website. This is a minimalist, clean portfolio showcasing my design work and expertise.

## Overview

This portfolio features:
- **About Section** - Learn about my design philosophy and personal interests
- **Projects Showcase** - Featured projects including Deutsche Bank, PNC, Cloud, and Postnord
- **Skills & Tools** - Design tools and core competencies
- **Contact Information** - Get in touch with me

## How to Use

1. Open `index.html` in your web browser to view the portfolio
2. Navigate using the sticky header menu
3. Customize the content in `index.html` with your own information
4. Modify colors in `styles.css` under `:root` CSS variables

## Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1a1a1a;
    --accent-color: #0066cc;
    /* ... other colors ... */
}
```

### Update Projects
Edit the project cards in the `<section id="projects">` section of `index.html`

### Add Social Links
Update the contact section in `index.html` with your social media URLs

## Deployment

### Deploy to GitHub Pages
1. Push this repository to GitHub
2. Go to repository Settings → Pages
3. Select the branch to deploy (usually `main`)
4. Your portfolio will be live at `https://yourusername.github.io/Rakesh`

### Other Deployment Options
- Netlify
- Vercel
- Any static hosting service
